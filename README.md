## Setup

Add this to your `build.gradle` file

```groovy
allprojects {
	repositories {
		...
		maven { url "https://jitpack.io" }
	}
}
```
```java
dependencies {
	com.github.esafirm.androidcommon:recyclerview:x.y.z
	com.github.esafirm.androidcommon:app:x.y.z
}
```
